# kwm-game

## specs
Korneliuszu,
Tak jak rozmawialiśmy w piątek, prosiłbym o utworzenie gry terenowej z możliwością wykonywania zdjęć na wydarzenie 2 kwietniu. Oto kryteria wykonania:
	•	ładne - jak berael FUTu (międzynarodowa konferencja fajnie byłoby się zaprezentować)
	•	kilka zadań, każde ma ogólny opis miejsca, podpowiedź do kliknięcia oraz możliwość wykonania zdjęcia
	•	w aplikacji powinna być dostępny leaderboard oparty na czasie od odkrycia zagadki do zrobienia zdjęcia
	•	cały interfejs w języku angielskim
	•	użytkownicy mogą przeglądać zdjęcia innych drużyn
	•	ładne proszę
	•	ładne i funkcjonalne
	•	prosiłbym o wykonanie aplikacji do 2 marca, czy jest to realny termin?
