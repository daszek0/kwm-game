Field game, with photo taking capability.
## Specs
- Mobile only app
- Map with tasks localizations
- Holds Tasks with its geolocations with hint to click and option to send photo to task
- Live leaderboard which shows current player scores based on task completion time
- There's teams, each team can see other people scores and photos
- Team based accounts
## Views
- Main View Game Info, team tasks completed, score and event time left
- Map view with marked tasks localisation
- Leaderboard view with all teams points info, you should have an option to click team and see their photos (for only tasks that your team completed)
## Tech
- uses express backend for data retrival
- Uses material UI for components styles
